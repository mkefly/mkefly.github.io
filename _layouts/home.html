Coronavirus mortality estimates

Description

We study the impact of the COVID-19 pandemia by analysing the data from the John Hopkins catalogue. We produce dynamical and predictive models of the 'number of deaths' by calibrating the data to two sigmoidal curves ( Logistic and Gompertz ) using Bayesian parameter estimation tools. This interface is a rapid applet to produce predictions for countries with at least 140 fatalities, countries where few data points are available this uncertainty grows. We will focus on the number of deaths as it is a more robust value; given that it is not subject to the different way of testing between the countries. Data and plots are updated in a daily basis. Strict confinement measures are essential to flatten the curves predicted by both models. {{ content }}
